# Task Manager App

## Project Explanation
This is a simple Task Manager Web App built using HTML, CSS, and JavaScript.

The application allows users to:
- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed

The app provides a clean and user-friendly interface to manage daily tasks efficiently.

## Features
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Complete
- Responsive and Clean UI

## Tech Stack Used
- HTML
- CSS
- JavaScript

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in browser
